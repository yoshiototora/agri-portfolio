Background & Vision

実際に見て、考えて、つくる

キウイ農家での農作業を体験したとき、実際にやってみないと分からないことがたくさんあると感じました。

作業の大変さだけでなく、アプリの文字が見づらいといった小さな困りごとも、実際に足を運んだからこそ気づくことができました。

それ以来、まずは自分で見たり体験したりするようになりました。

原体験：キウイ農家での農作業体験

2026年5月、農業のデイワークに参加し、キウイ農家での農作業を体験しました。そこで、消費者からは見えにくい多くの地道な工程があることを知りました。

受粉作業と学び： 実際に農作業を体験する中で、消費者からは見えない地道な手作業の積み重ねが品質を支えていることを実感しました。この体験をきっかけに、自分でも家庭菜園を始めました。実際に育ててみると、何をいつやればよいのか分からないことも多く、生産者だけでなく、これから始める人にもさまざまな悩みがあることを知りました。

実際にやってみて分かったこと：人手不足と画面の使いづらさ

お世話になった高齢のご夫婦農家は、人手不足のためデイワークを利用していました。しかし、直前のキャンセルが多く、「経験の有無よりも、予定通り人が来てくれること自体がありがたい」という言葉が特に印象に残っています。人手不足が単なる効率の問題ではなく、生産活動そのものを続けるのが難しくなる深刻な問題であることを実感しました。

また、高齢の農家から「アプリの文字が小さくて見づらい」という声を聞きました。どんなに優れた機能があっても、使う人が使えなければ価値にならないことを身をもって学びました。この経験が、文字サイズ変更やUDフォントなど、誰でも使いやすい設計を重視するきっかけになっています。

防犯ボランティア / チームCG制作

フィッシング詐欺防止 3DCGアニメーション

警察の防犯ボランティアにおける若年層向け啓発映像制作（全国1位最優秀賞）

役割・担当: ディレクター兼メインクリエイター / 3名有志チーム
技術スタック: Blender / Adobe Premiere Pro / Git
警察庁 受賞詳細 長崎大学 NEWS掲載 共同通信 記事 読売新聞 記事 NBC長崎放送 記事 YouTube（完成映像）

💡 概要

若年層にフィッシング詐欺の危険性を30秒で直感的に伝える啓発アニメーションです。

⚠️ 背景

警察のサイバー防犯ボランティア活動（ネットパトロール等）に参加する中で、自分と同世代の大学生や若者がフィッシング詐欺で簡単に騙されていく現実を知りました。従来のポスターやチラシなどの啓発物では若者に読まれないため、身近なSNSで見かける動画広告のような形式で、自分事として危険性に気づく映像を作れないかと考えました。

🛠️ 取り組み

若者が普段スマホで動画をスクロールする様子を意識し、親しみやすいキャラクターから始まり、途中で動画が巻き戻って「詐欺のポイント」を解説する30秒の構成をディレクションしました。映像内の偽サイト画面はHTML/CSSで自作して撮影し、本当にある偽画面の質感を再現しました。チーム内では全体の方向性を提案し、BGMや背景などの役割を分担しながら制作を進めました。

🌱 結果

警察庁主催コンテストで全国1位（サイバー警察局長賞）を受賞しました。身近な友人からも『かわいらしい見た目なのにしっかり怖さが伝わる』と好評を得て、狙い通り『見やすさと危険性の両立』を届けることができました。

📝 学び

単に迫力ある映像を作るのではなく、若年層の実際のネット利用体験に基づき『自分ごと』として危機感を持ってもらうストーリー設計を行うことで、初めて伝えるべきメッセージが相手に届くことを学びました。

大学院研究 / 修士課程（個人研究）

AIを用いた360度3D立体映像の境界破綻低減

少ないレンズ数と軽量な処理で、つなぎ目のズレが少ない360度立体映像を生成する研究

役割・担当: 研究者（Unity実装、AIアルゴリズム構築、物理器具自作）
技術スタック: Python / PyTorch / OpenCV / Unity / 3Dプリンター
修士研究メインコード 立体映像評価用(vr-ssq-evaluation) Quest 3S 表示テスト用

💡 概要

高価な専用機材を使わず、市販の360度カメラで高品質な360度3D映像を生成することを目指しています。学部の卒業研究で見えてきたつなぎ目のズレや処理速度の問題を、AI合成技術を用いて改善する研究です。

⚠️ 背景

自分で撮影した360度映像をVRゴーグルで見ると、つなぎ目にズレ（境界破綻）があり不自然さを感じました。このズレをなくすには通常、高価な専用カメラ機材が必要であり、また撮影時のカメラの手ブレや揺れによって映像のズレがさらに悪化してしまう点が課題でした。

🛠️ 取り組み

少ないレンズ数の安価な機材構成でも綺麗に合成できるよう、AIを用いて映像のつなぎ目のズレをリアルタイムで検出し、自動補正する合成アルゴリズムを開発しました。また、撮影時の物理的なブレを根本から防ぐため、3Dプリンターでカメラ固定用のアタッチメントを自作し、プログラムと物理器具の両面からアプローチしました。

🌱 結果

1フレームあたり約3秒かかっていたAIによる映像合成処理を高速化し、リアルタイムでの配信・生成を可能にしました。被験者による検証を通じ、安価な機材構成でもつなぎ目のズレや違和感を抑えた立体映像を生成できることを確認しました。

📝 学び

簡易360°全周囲3D映像のシームレス合成手法

安価なカメラ3台を用いた低コストで自然な360度立体VR映像の再現

役割・担当: 卒業研究者（個人研究・プログラム設計・器具作成すべて）
技術スタック: Python / PyTorch / OpenCV / 3Dプリンター
卒業研究のメインコード

💡 概要

市販の360度カメラ3台を組み合わせ、低コストで自然な360度立体VR映像を合成する研究です。

⚠️ 背景

市販の安価な360度カメラを複数使って立体映像を撮影すると、映像のつなぎ目で人の体が途切れて見えたり、映像が不自然に揺れたりする境界破綻（つなぎ目のズレ）が起きていました。また、従来の固定的な補正方法では、被写体の動きや撮影場所の明るさの変化に柔軟に対応できない点が課題でした。

🛠️ 取り組み

映像の中の動く人物と背景を自動で区別し、つなぎ目を滑らかにつなぎ合わせるフェード処理と合成手法を開発しました。プログラムだけでなく、3Dプリンターを用いてカメラを正しい位置で物理的に固定する専用台も自作し、ハードとソフトの両面からズレを最小限に抑えました。

🌱 結果

つなぎ目のズレや違和感を大幅に抑えた、自然な360度立体映像の合成に成功し、卒業論文として発表しました。この研究を通じて、安価な機材でも工夫次第で高品質な合成ができる実証を得て、さらなる軽量化とリアルタイム処理を目指して大学院での研究へ繋げました。

📝 学び

高価な専用機材を用意しなくても、自分で工夫してプログラムと固定台を組み合わせることで、つなぎ目のズレといった違和感を大幅に減らせることを体験し、自ら手を動かして工夫を重ねる面白さを学びました。

個人開発 / iOSアプリ配信

iOSアプリ「HabitSpark」の開発と改善サイクル

一度のできなかったことで挫折しない「失敗しても続けられる仕組み」と、レビューを反映した改善

役割・担当: 個人開発（企画・UI/UX・DB設計・実装・配信・アップデート保守すべて）
技術スタック: Swift / SwiftUI / SwiftData / Git
App Store (HabitSpark) GitHub (GoalTracker)

💡 概要

「HabitSpark」は、挫折を防ぐための習慣化支援iOSアプリです。企画、デザイン、Swiftでの実装、App Storeでのリリースまで一貫して単独で行いました。

⚠️ 背景

多くの習慣化アプリは毎日続けることが前提で、一度できなかっただけで続ける気持ちが切れてしまったことがありました。また、大きな目標を立てても、何から始めればよいのか分からず行動に移せないこともありました。そこで、「失敗しても続けられること」と「次に何をすればよいかが分かること」を大切にした習慣化アプリを作ろうと考えました。

🛠️ 取り組み

目標を月・週・日の3段階に分解できるようにし、失敗した日はKPT（Keep/Problem/Try）で振り返れる仕組みを作りました。また、振り返りで考えた改善策（Try）がそのまま翌日のタスクとして登録されるようにし、自然に改善サイクルが回るよう設計しました。

🌱 結果

App Storeのレビューで「未完了タスクを翌日に引き継ぎたい」という声を受け、単純なコピーではなく、繰り返し未完了になるタスクは目標設定自体を見直せる仕組み（3日連続未完了で自動的にProblemへ戻す仕組み）を実装しました。

📝 学び

作って終わりではなく、実際に使われて初めて見えることがあると学びました。特に、ユーザーからのレビューをきっかけに機能を改善した経験から、自分の想定だけでなく、実際の利用状況を見ながら継続的に改善することの大切さを実感しました。
